// about

A community of cloud security practitioners in Pakistan.

Independent. Run by engineers. CNCF-affiliated. We are 280+ members across Lahore, Karachi, and Islamabad — building a body of cloud-native security practice that didn't exist here before. No gatekeeping. No corporate cringe. Just the work.

// mission

Make cloud security knowledge accessible to every engineer in Pakistan.

We run sessions, publish walkthroughs, and build labs so practitioners at every level can ship secure infrastructure — regardless of city, company size, or credentials.

// vision

A Pakistan where cloud-native security is a practiced discipline, not a buzzword.

We want Pakistani engineers contributing upstream to CNCF security projects, speaking at KubeCon, and setting the standard for how cloud infrastructure gets hardened in this region.

// origin

Defense in depth. Hospitality on top.

CNSPK — Cloud Native Security Pakistan — was founded in August 2023 by a group of engineers who noticed the gap: Pakistan had cloud practitioners, but no dedicated space for the security side of cloud-native. No meetups dissecting Kubernetes RBAC misconfigs. No community reviewing supply-chain attestation. No mehfil for the people who actually read the CVE.

So we built one. Affiliated with the Cloud Native Computing Foundation as a recognised chapter, but independent in governance, voice, and direction. We run our own events, set our own agenda, and answer to the community — not a parent org's marketing calendar.
